Usage

This website aggregates data from several sources, such as AnimeNewsNetwork's encyclopedia API . To search, enter a keyword into the box above. It will search in a database of anime and manga titles to return works of interest. Note: if an anime title does not have a crunchyroll link associated with it, that does not necessarily mean that crunchyroll does not have it.
